## Ejercicio 2
```python
celsius = [0, 20, 37, 100]
def celsius_far(grado):
    return (grado * 9 / 5) + 32
res = list(map(celsius_far, celsius))
print(res)
```

## Ejercicio 4
```python
palabras = ["perro", "gato", "elefante", "oso", "jirafa"]
def contar_letras(palabra):
    return len(palabra) > 5
largas = list(filter(contar_letras, palabras))
def a_mayus(word):
    return word.upper()
res = list(map(a_mayus, largas))
print(res)
```
